Admin controls

Restricted to the organiser. Use this to refresh scores from the Google Sheet.

Must match the key configured in the Google Apps Script backend. It is not visible to other users.

Workflow on each update: paste the latest Masters leaderboard table into the RawLeaderboard tab in your Google Sheet, then click Refresh scores from sheet. The backend will parse scores, update the Scores tab, and timestamp the refresh.

Setup notes (for the organiser)

1. Create a Google Sheet with tabs named Field, Entries, Scores, RawLeaderboard, and Meta.

2. Paste the official Masters field and fractional odds snapshot from Paddy Power into Field (columns: Golfer name, Odds numerator).

3. Add the Google Apps Script backend to the Sheet, deploy as a web app, and paste the deployment URL into the API_BASE_URL constant below.

After that, everything (entries, validation, leaderboard, dashboard, score refresh) is driven via this UI and the Sheet; no further coding required.
